# How to run Rust Files:

### Finding the source code:

- See if your main.rs file is there, by using the ls terminal command,
`ls`, it'll should then output,
`main.rs`.
- If you don't see your rust file, you are probably in the wrong directory, use `cd` to navigate your project.

### Compiling the source code:

- In the same directory as your `main.rs` file, run `rustc main.rs` in the terminal.
- This will create a binary file, `main.exe` on Windows, and `main` on Mac an Linux.
- You can double check this by running `ls` in the terminal.

### Running the binary file:

- To run the file on Windows, type `.\main.exe`, or if your on Mac or Linux, type `./main` to run the file.
